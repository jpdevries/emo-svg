# Build

## Getting Started
After cloning the repository run `npm install` or `yarn` to install the `node_modules`. You can then build and test the project.

### Test emoSVG
```bash
grunt test
```

### Build emoSVG
```bash
grunt build
```

### Watch emoSVG
```bash
grunt watch #watches source files in _build/js for changes and build files
```
